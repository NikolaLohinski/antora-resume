<icon-fa code> Development

Getting started

  • Install NodeJS >= 14.15.1

  • Clone the repository with git

$ git clone https://github.com/NikolaLohinski/antora-resume

  • Install dependencies

$ npm install

Run

$ npm start

Build

$ npm run build
